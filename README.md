# mherzog4/homebrew-tap

Homebrew tap for [tandem](https://github.com/mherzog4/tandem) — a shared seat inside your coding-agent session.

```sh
brew install mherzog4/tap/tandem
```

`Formula/tandem.rb` is generated by
[`scripts/update-formula.sh`](https://github.com/mherzog4/tandem/blob/main/scripts/update-formula.sh)
in the main repo and refreshed on each release.
